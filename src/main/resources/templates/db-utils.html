🗄️ Database Utilities

💾 Save – Backup Database

Creates a complete dump of all database tables (images, course types, counters, trainers, lecturers, technicians, participants, courses, students, instructors) and downloads it as a .json.gz file (GZIP-compressed JSON, schema v1.0).

Binary data (photos, images) is stored as Base64 within the JSON. The file can be used later with the Restore function below.

🔄 Restore DB
⚠️ Warning: Restoring will permanently replace all current data with the contents of the backup file. This action cannot be undone.

Select a .json.gz backup file created by the Save function above. The schema version is validated before any data is changed. Expected schema version: 1.0.

ℹ️ Format details: GZIP-compressed JSON. Images and photos are Base64-encoded within the file.
The backup includes: images, course types, course counters, trainers, lecturers, technicians, participants, courses, students, instructors.
